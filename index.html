Pioneer-rec

Recovery program for Pioneer DVR-633H, DVR545H harddisks

View on GitHub Download .zip Download .tar.gz

So there you are with your Pioneer DVR633H-S or DVR545H showing you an HDD Error and refusing to boot... This usually indicates that the harddrive in the recorder went bad. But don't panic, there may be hope!

Unlike other utilities for this task, this program is capable of reconstructing the movies from the harddrive by parsing the filesystem structure of the Pioneer DVR instead of just dumping MPEG-chunks!

For further information on how to use it and for technical details about the filesystem structure, please refer to the README file. For the impatient ones, assuming you have a disk image image.dd and a recovery destination directory f:\dump, just do:

pioneer_rec -d c:\image.dd f:\dump
